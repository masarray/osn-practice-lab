# OSN Challenge Calibration v1.2

This update recalibrates the OSN portion of the bank after child testing showed the earlier questions were too easy for a Grade 5 student moving to Grade 6.

## What changed

- Added **300 harder OSN Challenge calibration items** across Matematika, IPA, and IPS.
- Each OSN subject now has 100 `osn-challenge` items.
- Updated simulation question selection so OSN Simulation prioritizes `osn-challenge` and `hard` questions first.
- Daily Mission remains lighter so the habit loop does not become intimidating.
- Public bank remains original practice content, not copied official OSN/ANBK papers.

## OSN calibration rule

For OSN subjects, the app should avoid simple recall-heavy items in the main simulation. Good OSN-style items should usually include at least one of these patterns:

1. Multi-step calculation or inference.
2. Trap distractors that match common mistakes.
3. Data/stimulus interpretation.
4. Reasoning from cause to effect.
5. A short explanation that teaches the method, not just the answer.

## Difficulty policy

- Daily Mission: mixed and friendly.
- Try Again: wrong-question first, difficulty follows user's history.
- OSN Simulation: challenge-biased; prioritize `osn-challenge`, then hard, then medium.
- SD curriculum / ANBK banks: retained for general skill foundation and literacy/numeracy practice.
